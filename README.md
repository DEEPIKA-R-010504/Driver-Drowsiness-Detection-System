# Driver-Drowsiness-Detection-System
Driver Drowsiness is a major cause of road accidents, resulting in fatalities and injuries.

According to National Highway Traffic Safety Administration(NHTSA),around 100,000 accidents are reportedeach year due to drowsy driving.

The purpose of this project is to develop a driver drowsiness detection model that uses data science and machine learing.

The system is desingned to monitor the driver's face and alert them if they show sign ofdrowsiness or fatigue.
