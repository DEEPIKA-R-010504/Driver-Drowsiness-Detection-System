# Driver-Drowsiness-Detection-System
PROBLEM STATEMENT: 

Driver drowsiness is a major cause of road accidents, resulting in fatalities and injuries. According to the National Highway Traffic Safety Administration (NHTSA), around 100,000 accidents are reported each year due to drowsy driving.

The purpose of this project is to develop a driver drowsiness detection system that uses data science and machine learning techniques. The system will be designed to monitor the driver's face and alert them if they show signs of drowsiness or fatigue.
