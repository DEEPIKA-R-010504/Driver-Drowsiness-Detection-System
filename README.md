# Driver-Drowsiness-Detection-System

Driver drowsiness is a major cause of road accidents, resulting in fatalities and injuries.

According to the National Highway Traffic Safety Administration (NHTSA), around 100,000 accidents are reported each year due to drowsy driving.

The purpose of this project is to develop a driver drowsiness detection model using data science and machine learning.

The system monitors the driver’s face and alerts them when signs of fatigue or drowsiness are detected.
